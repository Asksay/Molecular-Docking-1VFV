# 🧬 Molecular Docking of Curcumin with 1VFV

## 📖 Overview
This project demonstrates the molecular docking of Curcumin with the target protein **1VFV** using computational drug discovery tools.

## 🎯 Objective
To investigate the binding affinity and molecular interactions between Curcumin and the 1VFV protein through molecular docking.

## 🛠️ Software Used
- AutoDock 4
- AutoDockTools (ADT)
- Discovery Studio Visualizer
- PyMOL

## 🔬 Workflow
1. Protein Preparation
2. Ligand Preparation
3. Grid Generation
4. Molecular Docking
5. Protein–Ligand Interaction Analysis

## 📂 Project Files
- Protein (.pdb, .pdbqt)
- Ligand (.pdbqt)
- Grid Parameter File (.gpf)
- Docking Parameter File (.dpf)
- Docking Log File (.dlg)
- Grid Map Files (.map)

## 📊 Results
The docking study was performed successfully using AutoDock 4. Binding affinity and protein–ligand interactions were analyzed for the best docking pose.
